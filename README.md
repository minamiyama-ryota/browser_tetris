# ブラウザ テトリス

HTML + CSS + JavaScript のみで動作するブラウザ版テトリスです。外部ライブラリは一切不要です。

## 遊び方

`index.html` をブラウザで開くだけでゲームが始まります。

## 操作方法

| キー | 動作 |
|------|------|
| ← （左矢印） | ブロックを左に移動 |
| → （右矢印） | ブロックを右に移動 |
| ↑ （上矢印） | ブロックを時計回りに回転 |
| ↓ （下矢印） | ソフトドロップ（加速落下） |
| スペース | ハードドロップ（即座に落下） |

## ゲームルール

- **ライン消去**: 横一列が全て埋まると消去されます。
- **スコア**: 同時に消去したライン数によりスコアが加算されます。
  - 1ライン: 100点 × レベル
  - 2ライン: 300点 × レベル
  - 3ライン: 500点 × レベル
  - 4ライン: 800点 × レベル（テトリス）
- **レベルアップ**: 10ライン消去するごとにレベルが上がり、落下速度が上昇します。
- **ゲームオーバー**: ブロックがフィールド上部に積み上がると終了です。「もう一度プレイ」ボタンでリスタートできます。

## テトリミノ（ブロック）の種類

| 種類 | 形状 | 色 |
|------|------|-----|
| I | 4マス直線 | 水色 |
| O | 2×2 正方形 | 黄色 |
| T | T字型 | 紫 |
| S | S字型 | 緑 |
| Z | Z字型 | 赤 |
| J | J字型 | 青 |
| L | L字型 | オレンジ |
